# D-Basket (Blinkit Clone) 🛒

D-Basket is a high-performance grocery delivery application inspired by Blinkit. This project showcases modern Android development using Jetpack Compose and a clean UI/UX design.

## 📥 Download the App
Get the latest version of the app here:

---

## 🚀 Features
- **Fast Shopping:** A smooth, reactive UI for browsing groceries.
- **Categories:** Easy navigation through snacks, dairy, fruits, and more.
- **Modern Tech:** Built with **Jetpack Compose** for a fully declarative UI.
- **Dark Mode Support:** Designed to look great in any theme.

## 🛠 Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Image Loading:** Coil
- **Architecture:** MVVM (Model-View-ViewModel)
